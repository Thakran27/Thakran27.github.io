# thakran27.github.io

Academic portfolio of **Dr. Anjali Thakran** — postdoctoral researcher in semiconductor physics, perovskite photovoltaics, tandem solar cells, and optoelectronic devices at Academia Sinica.

**Live site:** https://thakran27.github.io/

## Features

- Interactive hero with a photon/charge-carrier physics simulation (click to emit a photon burst)
- Explorable tandem solar cell schematic — hover each layer to learn its role
- Year-by-year academic and research journey timeline
- Filterable publication list (all / first author / collaborations) with DOI links
- Animated skill meters, scientific-photography art cards with 3D tilt, and a physics-themed 404 page

## Tech

A single self-contained `index.html` — no frameworks, no build step. Vanilla HTML/CSS/JS with Canvas 2D and SVG animations. Fonts: Plus Jakarta Sans, Manrope, Space Grotesk (Google Fonts). Deployed with GitHub Pages.

## Editing

All content lives in `index.html`. Publications, timeline entries, skills, and awards are plain HTML blocks — edit and push to `main` to update the live site.
